# PSO Regnskap Website

Professional Norwegian accounting website for PSO Regnskap AS.

## Tech Stack
- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- shadcn/ui components
- Framer Motion animations

## Features
- Public website with 5 pages
- Admin dashboard (coming in Part 2)
